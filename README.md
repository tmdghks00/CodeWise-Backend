# CodeWise-Backend

**AI 기반 실시간 코드 분석 및 추천 서비스 - 백엔드**

CodeWise는 실시간 코드 품질 분석과 AI 기반 추천을 제공하는 웹 애플리케이션입니다.  
이 저장소는 Spring Boot 기반 백엔드 API 서버를 구성하며, 프론트엔드 및 AI 서버와 연동됩니다.

---

## 🔧 기술 스택
- Java 17
- Spring Boot 3
- Spring Security + JWT
- WebSocket (STOMP)
- MySQL (AWS RDS)
- AWS EC2 배포
- Swagger API Docs
